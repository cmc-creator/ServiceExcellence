# Pilot Outcomes Plan (Hospital/Facility)

Use this as the standard pilot runbook for first hospital/facility deployments.

## Pilot Goal

Demonstrate that the program improves completion quality, escalation accuracy, and manager follow-through, not just completion volume.

## Suggested Timeline (90 Days)

Weeks 1-2: Setup

- Confirm integration path (SCORM or xAPI).
- Configure org slug, roles, and access controls.
- Validate one end-to-end completion in sandbox.

Weeks 3-6: Controlled launch

- Start with 15-25 users across clinical, non-clinical, and leadership tracks.
- Run manager follow-up coaching within 7 days of completion.
- Capture learner confusion points and content friction.

Weeks 7-10: Expansion

- Expand to additional departments.
- Compare early cohort vs expanded cohort completion quality.
- Apply wording and routing adjustments.

Weeks 11-12: Decision package

- Produce pilot summary with outcomes and open risks.
- Confirm procurement blockers and required evidence.
- Present go/no-go recommendation.

## Pilot Success Metrics

Track at minimum:

- Completion rate by role track.
- Pass rate on final assessment.
- Category weakness trends (retry recommendations).
- Time-to-completion and repeated-attempt rate.
- Manager coaching completion rate (within 7 days).

## Baseline vs Post-Pilot Comparison

Capture a baseline before launch, then compare at day 30, 60, and 90:

- % of learners with completed annual requirement.
- % of learners requiring remediation.
- Top 3 missed domains by role track.
- % of managers completing documented follow-up.

## Required Artifacts for Buyer Review

- Pilot metrics dashboard export.
- Compliance control references (policy citation map + attestation).
- LMS integration evidence (event and completion flow).
- Manager follow-up evidence using toolkit.
- Known limitations and mitigation plan.

## Roles and Ownership

| Workstream | Owner |
| --- | --- |
| IT/LMS setup | Facility IT + NyxArete technical lead |
| Compliance review | Facility compliance + NyxArete content lead |
| Manager coaching rollout | Department managers |
| Analytics review | Program owner + ops sponsor |
| Final recommendation | Executive sponsor |

## Go / No-Go Criteria

Go when all are true:

- Completion and pass metrics meet target thresholds.
- No critical integration failures remain.
- Compliance sign-off is complete.
- Buyer evidence packet is ready for procurement.

No-Go / Extend when any are true:

- Critical tracking fields are missing or unreliable.
- Manager coaching adoption remains low.
- Policy mapping or attestation language is not approved.
