# Content Intake Template for Existing Decks

Use this template while reviewing both PowerPoints.

## Slide Tagging Fields
- Source deck name
- Slide number
- Topic area (Service Excellence or Code of Conduct)
- Keep / Merge / Rewrite / Retire
- Policy owner (HR, Compliance, Clinical Ops)
- Interaction opportunity (quiz, branching, timed, game)
- Humor-safe opportunity (yes/no)
- Notes

## Priority Rules
- Keep scenario-based slides with clear behavior outcomes.
- Rewrite text-heavy legal content into short explainers plus reference links.
- Retire duplicate or redundant concept slides.
- Ensure every required policy concept appears at least once in an interactive moment.

## Final Deliverables
- Final content map by module
- Scenario script pack
- Assessment question bank
- Policy appendix with version/date
